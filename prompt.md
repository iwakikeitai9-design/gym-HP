■ 全体の雰囲気
「親しみやすく温かみがあり、それでいて力強さと本気度も感じるデザイン」
小樽という地方都市の地に根ざしたジムらしい、誠実さと情熱が伝わるトーンにすること。
コーポレートサイトのような堅さではなく、人の顔が見える温かいホームページにすること。

■ カラーパレット（css/style.css の :root でCSS変数として一元管理すること）
  --color-primary:   #C0392B  /* メインカラー：情熱の赤 */
  --color-secondary: #E67E22  /* サブカラー：温かみのあるオレンジ */
  --color-bg:        #FDF6EC  /* 背景：やわらかいクリーム */
  --color-text:      #2C2C2C  /* 本文テキスト：ほぼ黒 */
  --color-cta:       #27AE60  /* CTAボタン（行動を促すボタン）：緑 */
  --color-white:     #FFFFFF
  --color-gray:      #F5F5F5
  --color-dark:      #1A1A1A  /* フッター背景等 */

■ フォント（Google Fontsを使用）
  見出し：Noto Serif JP（重厚感・信頼感）
  本文　：Noto Sans JP（読みやすさ重視）
  ※Google Fontsのlinkタグを各HTMLの<head>内に必ず記述すること

■ レスポンシブ対応（画面幅に応じた表示切り替え）
  スマートフォン：〜767px
  タブレット　　：768px〜1023px
  PC　　　　　　：1024px〜

■ アニメーション・インタラクション
  - スクロールで画面に入ったとき、セクションがふわっとフェードインする
    （Intersection Observer APIを使用）
  - ボタンにホバー（マウスを乗せた）したとき、色が滑らかに変わる（transition: 0.3s）
  - ヒーロー画像のキャッチコピーが、ページ読み込み時にゆっくりフェードインする
  - ヘッダーはページ最上部では背景透明、スクロールすると白背景＋影が出現する
  - ページトップへ戻るボタンが、300px以上スクロールすると右下に表示される
  - ページ読み込み時に0.8秒程度のローディング画面を表示する

【JavaScript仕様】
━━━━━━━━━━━━━━━━━━━━━━━━━━━
- jQueryは使わず、バニラJS（ライブラリなしの素のJavaScript）を基本とする
- 必要な場合はCDN経由のライブラリを使ってよい
- Swiper.js（スライダー用）は必ずv11を使用すること
    CDNリンク（各HTMLの</body>直前に記述）：
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.css">
    <script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-bundle.min.js"></script>

- main.js に実装する機能：
    1. ハンバーガーメニュー（スマホ時のナビゲーション開閉）
    2. スクロールアニメーション（Intersection Observer APIで要素の出現を検知）
    3. ヘッダースクロール追従（透明→白背景＋影）
    4. FAQアコーディオン（about.html：クリックで開閉・CSSのmax-heightで実装）
    5. フォームバリデーション（contact.html：未入力チェック・赤文字でエラー表示）
    6. フォーム送信処理（送信ボタンを押したとき preventDefault でページリロードを止め、
       「送信が完了しました。2営業日以内にご連絡します。」というメッセージを表示する）
    7. スムーズスクロール（ページ内リンクをクリックでゆっくり移動）
    8. トレーナー紹介スライダー（Swiper.js v11を使用）
    9. ページトップへ戻るボタン（300px以上スクロールで表示・非表示）
   10. ローディング画面（ページ読み込み時0.8秒のアニメーション）
   11. お知らせ一覧のカテゴリー絞り込みフィルター（news.html）

■ グローバルナビゲーション（ヘッダー）
  - 左側：ジム名ロゴ「Desafio」（テキスト。フォントはNoto Serif JP・太字）

  - ページ最上部では背景透明・文字色は白
  - スクロールすると白背景＋影・文字色はダークに変わる
  - 現在表示しているページのリンクに下線またはハイライトを付ける（アクティブ表示）

    該当の入力欄の下に赤文字でエラーメッセージを表示する
- 送信処理：
    送信ボタンを押したとき preventDefault でページリロードを止め、
    フォームの上部に「✅ 送信が完了しました。2営業日以内にご連絡します。」
    というメッセージをアニメーション付きで表示する
- 送信ボタン：「送信する」（--color-cta の緑・幅いっぱい）
- フォームのaction属性は "#" にして、以下のコメントを明記：
    <!-- ← メールフォームサービス（FormspreeやNetlify Forms等）のURLをここに入れてください -->

    住所・電話番号・営業時間（← 後から変更してください）
- SNSフォローボタン：Instagram・X（Twitter）
- 「よくある質問はこちら →」リンク（about.html#faq へ）
